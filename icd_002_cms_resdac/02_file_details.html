File Details

This interface covers multiple CMS claim-level and beneficiary-level datasets provided via ResDAC. Each table below lists the column name, a short description, and the source data type/length as received. Field names are preserved from source and may contain underscores or CMS abbreviations. All personally identifiable information (PII) or protected health information (PHI) must be handled according to State and Federal policy.

Consists of Medicare Fee-for-Service Research Identifiable Files (RIFs) obtained through CMS State Agency Request authority. These files contain detailed Medicare claims and enrollment information for beneficiaries in Hawaiʻi and support longitudinal analyses of healthcare access, quality, utilization, and outcomes among older adults and individuals with disabilities.
